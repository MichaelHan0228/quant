# ETF轮动策略 - 完整说明文档

## 1. 策略概述

### 1.1 策略目标
通过量化方法，在5类大类资产ETF之间进行动态轮动，追求：
- 年化收益10%+
- 最大回撤<15%
- 夏普比率>0.8

### 1.2 ETF池
| ETF | 代码 | 资产类别 | 特点 |
|-----|------|----------|------|
| 黄金ETF | 518880 | 避险资产 | 与其他资产低相关 |
| 红利ETF | 510880 | A股价值 | SmartBeta，低波动 |
| 标普500ETF | 513500 | 美股宽基 | QDII，有溢价风险 |
| 恒生科技ETF | 513180 | 港股科技 | QDII，高波动 |
| 沪深300ETF | 510300 | A股宽基 | 流动性最好 |

### 1.3 核心逻辑
```
每周五收盘后：
  1. 计算每个ETF的"风险调整动量"
  2. 筛选趋势向上的ETF（双均线确认）
  3. 按动量排名，选前2-3个
  4. 相关性过滤，避免高相关ETF同时持有
  5. 根据波动率动态调整持仓数

下周一开盘：
  按目标权重执行交易
```

## 2. 信号体系

### 2.1 多周期动量（核心信号）

**公式：**
```
动量 = 0.2 × 5日收益率 + 0.5 × 20日收益率 + 0.3 × 60日收益率
```

**设计思路：**
- 短期(5日)：捕捉近期拐点，但噪声大，权重低
- 中期(20日)：主信号，最核心，权重高
- 长期(60日)：确认大方向，权重中

### 2.2 风险调整动量（选股权重）

**公式：**
```
风险调整动量 = 多周期动量 / 年化波动率
```

**设计思路：**
- 纯动量会偏向高波动资产（如恒生科技）
- 除以波动率后，选出"每单位风险获得最高动量"的ETF
- 类似夏普比率的思路

### 2.3 双均线趋势确认

**逻辑（三重确认）：**
```
趋势向上 = (收盘价 > MA25) AND (MA25 > MA60)
```

## 3. 风险控制

### 3.1 三层止损体系

| 层级 | 触发条件 | 执行时机 | 作用 |
|------|----------|----------|------|
| 第一层 | 个股从成本价跌8% | 每周五 | 控制单票亏损 |
| 第二层 | 组合从高点回撤10% | 每周五 | 控制组合风险 |
| 第三层 | 本周某日暴跌5%+反弹<50% | 每周五检查，周一执行 | 应对暴跌 |

**第一层：个股成本价止损**
```python
if (当前价 / 买入价 - 1) < -8%:
    卖出该ETF
```

**第二层：组合高点回撤止损**
```python
if (当前净值 / 历史最高净值 - 1) < -10%:
    所有持仓减半（不是清仓）
```

**第三层：暴跌后反弹止损（改进版）**
```python
# 周五收盘后检查本周数据
# 找出最大单日跌幅那天（暴跌日）
# 计算反弹程度
recovery = (周五收盘价 - 暴跌日最低价) / (暴跌日开盘价 - 暴跌日最低价)

if 本周某日跌幅 > 5%:
    if recovery >= 50%:
        不止损（反弹足够）
    else:
        止损（反弹不足）
```

**设计思路：**
- 暴跌后如果完全反弹回去，说明市场消化了利空，不必止损
- 暴跌后只反弹一半，说明买盘不足，应该止损
- 避免在反弹高点卖出（事后止损的改进）

### 3.2 波动率择时

**逻辑：**
```
用沪深300的20日年化波动率作为市场风险指标

if 波动率 > 90%历史分位:
    空仓（极端风险）
elif 波动率 > 80%历史分位:
    最多持1个ETF（高风险）
else:
    正常持仓2-3个
```

### 3.3 动态持仓数

| 市场状态 | 条件 | 持仓数 |
|----------|------|--------|
| 极弱 | 波动率>90%分位 | 0（空仓） |
| 弱 | 波动率>80%分位 | 1 |
| 中 | 1-2个趋势向上 | 1-2 |
| 强 | 3+个趋势向上且动量强 | 3 |

## 4. 交易执行

### 4.1 时序说明

```
周五收盘后 → 计算信号（用周五收盘价）✓ 合理
周一开盘前 → 检查止损条件（用本周数据）
周一开盘   → 执行交易（用周一开盘价）
```

### 4.2 避免数据穿越

- 信号计算：用周五收盘价（周五收盘后能看到）
- 仓位计算：用周一开盘价（周一执行时的价格）
- 止损检查：用本周一~周五的数据（周五收盘后能看到）

### 4.3 真实费率

| 费用项 | 普通ETF | QDII ETF |
|--------|---------|----------|
| 佣金 | 万1.5（最低5元） | 万1.5（最低5元） |
| 买卖价差 | 2 tick (0.002元) | 3 tick (0.003元) |
| 印花税 | 0 | 0 |

### 4.4 部分调仓

只操作差异部分，减少不必要的交易。

### 4.5 信号触发式调仓

如果本周持仓代码和上周一样，不调仓。

## 5. 相关性过滤

### 5.1 问题
恒生科技和标普500有时高度相关，同时持有时分散化效果有限。

### 5.2 解决方案
```
计算60日滚动相关性
if 相关性 > 0.7:
    不同时持有
    选动量次高的ETF替代
```

### 5.3 QDII偏离度检查
QDII ETF盘中价格可能偏离实际净值。

```
偏离度 = (价格 - MA20) / MA20
if 偏离度 > 5%:
    可能溢价过高，降权50%
```

## 10. 风险提示

1. **回测不等于实盘**：历史表现不代表未来收益
2. **数据局限**：只有3年数据，未经历完整牛熊周期
3. **QDII溢价风险**：标普500/恒生科技ETF可能有溢价
4. **流动性风险**：极端行情下ETF可能有价差扩大
5. **过拟合风险**：虽然参数敏感性分析显示稳健，但仍需持续观察
