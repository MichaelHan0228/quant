# 全天候 + 股债性价比(ERP)择时（v2 叠加层）

> 本文档是 strategy-plan-v2.md 的**可选叠加层**，不改变 v2 的主体结论。
> v2 证伪的是"价格型择时"（回撤降仓，来回打脸）；本层是"估值型择时"，
> 信号慢、参数少、只调股债相对配比，回测验证有效但增益边际。

## 一、信号定义

```
ERP = 沪深300盈利收益率(1/PE_TTM) − 10年期国债收益率
分位 = 当前 ERP 在过去 5 年滚动窗口中的分位数
```

用 5 年滚动窗口而不用全历史：10 年前国债 4%、现在 1.7%，
利率中枢下移会让全历史分位系统性地喊"股票便宜"。

## 二、执行规则（叠加在 v2 五条规则之上）

1. **季度检查日**先按 ERP 分位更新股票腿 hlb 目标权重：

   | ERP分位 | hlb 调整 | bond10 反向 |
   |---|---|---|
   | ≥80% | +10pp | −10pp |
   | 60~80% | +5pp | −5pp |
   | 40~60% | 0 | 0 |
   | 20~40% | −5pp | +5pp |
   | <20% | −10pp | +10pp |

2. **单次步进 ±5pp**：目标从当前位置向档位方向每次最多挪 5pp，防边界横跳。
3. 只动 hlb/bond10 相对配比，黄金/豆粕/现金/标普500 不动；总权益敞口由 v2 版本决定。
4. 之后沿用 v2 阈值纪律：任何资产偏离目标 ≥±5pp 才调仓，不触发不动。

## 三、数据源（免费、无 key）

- 沪深300 PE_TTM 日频：中证指数官网 `index-perf`（字段 peg 实为市盈率），2014 至今
- 10年国债收益率日频：东财中债收益率曲线 `RPTA_WEB_TREASURYYIELD`（EMM00166466）
- 缓存：`data/csi300_pe.csv`、`data/cgb10y.csv`（7 天内新鲜则读缓存）

### 择时做对了什么 / 做错了什么

- ✅ 2020-03（分位95%）：疫情底加股票 → 吃修复
- ✅ 2020Q4~2021Q1（分位4%）：减到最低档 → 回避 2021 核心资产杀估值
- ✅ 2022-09（分位99%）起持续高配 → 吃 2023~2024 红利+反弹（2024 +18.7% vs 基线 +16.6%）
- ❌ 2021 全年低配股票：当年红利微涨，少赚约 1pp（+5.3% vs +6.4%）
- ❌ 2022 越跌越加：当年股票仍跌，多亏约 0.9pp——估值择时不能防当年下跌，只能靠 bond10 对冲

## 五、诚实声明

- **2020 起短样本里增益是边际的**：年化 +0.0~0.2pp、回撤改善 0.5~0.6pp；
  **2015 起长样本里明显更值**：年化 +0.3~0.5pp、回撤改善 2.6~4.0pp（见附录B）。
  短样本大顶大底只有 2021 顶和 2024 底两次，长样本多出 2015 泡沫顶和 2018 底，
  择时的真实价值恰恰在这类极端年份。
- 参数仅 3 个（5档分位阈值、±10pp 幅度、±5pp 步进），过拟合空间有限；
  但样本期利率单边下行，5 年滚动分位尚未经历过完整加息周期考验。
- 调仓 3→12 次（2020起）/ 4→24 次（2015起）、费用约 1100 元/11.5 年，成本可忽略。
- 若你不想每季度看一次信号，直接用 v2 基线也完全成立——这正是 v2 的设计初衷。

## 六、运行方式


# 附录C：DY股息率分位择时 与 "或"规则合并（ERP+DY）

> 思路：DY（红利低波股息率）处历史高位 = 红利便宜 → 调高 hlb。
> "或"规则：合并档 = max(ERP档, DY档)——任一信号说便宜就调高，两个都说贵才调低。

## 三个信号各自的脾气（从年度收益读出来的）

- **ERP 的强项在系统性极端年份**（2015 泡沫顶 +9.1% vs 基线 +6.3%；2018 +1.8% vs −0.4%），
  因为它看的是"股 vs 债"的大格局；弱项是 2021（红利便宜但国债收益率也高，ERP 看不清
  结构性便宜 → 踏空，+5.5% vs 基线 +6.9%）。
- **DY(5年分位) 漏掉 2015**：2015 泡沫在成长股，红利股没怎么涨，股息率没在 5 年窗口里
  打到极值 → 2015 年表现=基线。但 DY 看得到 2021 的结构性便宜（2021 +7.5%，补上 ERP 的坑）。
- **DY(全历史分位) 两个都能看见**：2009 起的扩展窗口让 2015 年的低股息率显得极端
  （2015 +9.1% 与 ERP 相同），2021 年也坚持高配。股息率是自洽指标、无利率中枢漂移，
  全历史窗口反而比 5 年窗口更可靠。
- **或规则·全史 = 两者优点叠加**：2015 跟 ERP 躲顶、2018 跟 DY 抄底（+1.8%）、
  2021 跟 DY 高配（+7.5%）、2022-2024 与 ERP 同步满仓。稳健版年化 8.37%、回撤 −8.1%
  （与 ERP 并列最优）；进取版年化 9.41% 为全部版本最高，回撤 −10.3% ≈ ERP。

## 结论

- **推荐择时层：或规则·全历史分位**（ERP 5年分位 ∨ DY 全史分位，取乐观档）。
  收益最高、回撤不劣化、费用 ~800 元/11.5 年、调仓 15-17 次。
- 若只想要夏普极致（不在乎少 0.1pp 年化），稳健版可留 ERP 单信号。
- DY 信号原为 PE 镜像（分红率稳定性假设）。真实股息率日频源已落地
  （全收益/价格反解日分红，见附录E）：镜像偏差被证实（分红率上升期偏保守），
  **真实版保留为可选配置**——稳健版换用基本无回撤代价，进取版按回撤容忍选择。

## 仓位边界实验（−5pp 下限，已回退）

2026-07-23 曾把减仓下限从 −10pp 收窄到 −5pp（稳健 hlb 最低 15%、进取最低 25%）重测：

| 或规则全史 | −10pp（现配置） | −5pp（已否决） |
|---|---|---|
| 稳健 年化/回撤 | **8.37%** / −8.1% | 8.14% / −8.1% |
| 进取 年化/回撤 | **9.41%** / **−10.3%** | 9.14% / −13.1% |
| 进取版 2015 年 | **+9.5%** | +6.3%（低于基线） |

极端年份保护是择时层的主要价值来源，收窄下限等于自废武功，已回退为 −10pp。
代码中保留 `ADJ_MIN` 参数（`backtest_dy.py`）以便复测。

## 为什么失败

- **极值区可以持续很久**：2022-09 ~ 2024-09 分位连续 2 年 >85%，
  "短暂替换"实际变成 2 年长持——而这段时间恰是红利风格最强的两年。
- **红利的防御属性和 300 的高 beta 在不同极值区轮流占优**：急跌型极值（2020 疫情）
  300 弹性确实好；阴跌+结构分化型极值（2022-2023）红利一直赢。一个阈值区分不了两种极值。
- 教训与 v2 一致：**在慢变量信号上叠加交易动作，样本内只有一次成功（2020）一次失败（2022-24），
  2 个样本点证明不了任何规律，净期望为负时就不该加这条规则。**

代码保留在 `backtest_swap.py` 作为证伪记录，不进入策略文档正文。

## 结论：择时有效，不是仓位幻觉

- **ERP 版平均仓位 ≈ 基线（+0.3pp），增益几乎全是择时**：同仓位下年化 +0.5pp、
  回撤 −10.9%→−8.1%，仓位没动而收益/回撤双改善。
- **或规则增益分解（稳健）**：总增益 +0.62pp 年化 = 仓位效应 +0.26pp + 择时 +0.36pp。
  仓位效应存在，但择时效应更大。
- **回撤改善 100% 来自择时**：固定高仓位对回撤无帮助（稳健 −10.7% 持平），
  进取版固定 32.9% 仓位回撤反而恶化到 −15.6%；只有择时压到 −8.1%/−10.3%。
- 实用推论：不想看信号可永久加权 hlb +3pp，拿走约四成收益增益、零调仓——
  但回撤一点不少，**加权和择时不是替代品**。
- 口径：平均仓位按目标权重路径时间平均；极端年份样本仍只有 2-3 个。

# 附录E：真实股息率信号落地（TTM分红/当下价格）—— 真实版保留为可选配置（2026-07-23）

> 动机：附录C 的 DY 是 PE 镜像（假设分红率稳定）。2023 年后分红率趋势性上升，
> 镜像系统性偏保守。现用中证官方数据推出真实股息率日频序列并验证。

## 数据方法（零新依赖，csindex 同源）

曾先探测 QMT/xtquant 取数：行情库不含 H30269/H20269（H 系列中证指数无代码、无权重、
无指数估值接口），ETF 分红因子 ≠ 指数股息率——**QMT 此路不通**，仍走 csindex。

```python
# 全收益指数 = 价格指数 × 累计分红再投因子，两指数日收益差即当日分红：
div_t = PX_{t-1} * (TR_t/TR_{t-1} - PX_t/PX_{t-1})     # H20269全收益 / H30269价格
DY_t  = sum(div_{t-249..t}) / PX_t                     # TTM分红 ÷ 当下价格
```

分母是当下价格：价格暴涨 DY 立刻下降、暴跌立刻上升，无滞后。实现：`real_dy.py`。
序列 2010-01 起可用，当前（2026-07-23）5.02%、全历史分位 79.0%；
历史区间 1.33%（2010-01）~ 7.76%（2022-07）。

### 教训：第一版口径曾致 2015 失明（已废弃）

首版用 `(TR_t/TR_{t-250})/(PX_t/PX_{t-250})−1`（分红累积增速，隐含分母是一年前价格）：
分母滞后一年，2015 泡沫顶 TTM 分红来不及跟上价格，DY 虚高至 4.3~4.9%、分位 81~99%
喊"极便宜"，**满仓进股灾**（进取版回撤 −17.8%）。修正为日分红反解口径后，
2015-03 分位 38.1%（−5pp 减仓）、2015-09 分位 87.1%（+10pp 抄底），序列行为正常。
