# 红利低波ETF 条件定投回测

## 策略规则

- 标的：红利低波ETF易方达（563020，跟踪中证红利低波动指数 H30269，季度分红）
- 每周最后一个交易日判断：当前价 vs 上一年最后一个交易日收盘价（前复权/全收益口径）
- **v5 现行规则（双锚 + 不对称分档）**：基准取"去年收盘价"和"MA250"中偏离更大者；
  0 ~ +5% 区间买 5 千；-5%~0 买 1 万；-5~-10% 买 2 万；以此类推每档 +1 万（涨幅超 +5% 停手）
- 历史版本：v1 仅 <0 才买；v2 ±5% 都买 1 万；v3 年末锚+0~5%买5千；v4 纯MA250锚（效果最差）。
  备份见 output/*_v1~v4.csv，当前输出为 v5
- 100 万资金池封顶，现金耗尽停投；季度分红现金回池参与后续定投

v5（双锚=年末锚和MA250锚取最大偏离）达到设计目标：保住v3的深跌力度（2020年买入70.8万），
又获得v4的牛市上车能力（2023/2024年买入8.9/6.6万）。XIRR 7.74%与最优的v2（7.87%）基本持平，
回撤相同——双锚没有进一步增益，但验证了"年末锚负责力度、均线锚负责上车"的机制成立。
最终结论：**追求收益选v2或v5，追求低回撤选v1**。

- 派发比例 0.7 用 2024/2025 真实分红校准（真实年派发约 3.8~3.9% vs 隐含股息率 5.1~5.9%）

## 实操部署（每周 5 分钟）

**平时**：全部闲置资金持有 511880 银华日利（场内货基，~2% 年化，T+0）。

**每周五 14:50**：
1. 查 563020 当前价和去年收盘价（每年 1 月更新一次基准）/ MA250，取两者偏离更大者
2. 算偏离度定档位：0~+5% → 5千；-5~0% → 1万；-5~-10% → 2万；每再低5%加1万；涨超+5% → 不操作
3. 卖出对应金额 511880（T+0 资金立即可用）
4. 买入 563020（100 份整数倍，佣金万1.5最低5元）

**分红到账日**：红利分红到账后当日买入 511880 停泊，下次定投一并使用。

**券商确认项**：① 511880 免佣金（主流券商均免）；② 最低 100 份（约 1 万元）。
